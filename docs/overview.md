# 프로젝트 목표 (Project Overview)

## 프로젝트 개요

**solo-agent**는 LangGraph 기반의 멀티 에이전트 시스템으로, 글로벌 배터리 시장에 대한 전략적 분석 보고서를 자동으로 생성합니다.

## 핵심 목표

1. **시장 분석 자동화**
   웹 검색과 뉴스 수집을 통해 글로벌 배터리 시장의 규모, 성장률, 지역별 수요, 원자재 가격 동향을 자동으로 수집·분석합니다.

2. **기술 역량 비교**
   RAG(Retrieval-Augmented Generation) 기반으로 기업별 기술 문서(특허, 기술보고서)를 검색하고, 배터리 셀 기술·소재·BMS·차세대 기술 역량을 평가합니다.

3. **경쟁사 전략 비교**
   LG에너지솔루션과 CATL을 대상으로 기술 전략, 지역 전략, 고객 전략, 원가 전략, 신사업을 비교 분석합니다.

4. **SWOT 분석**
   각 기업의 강점(S)·약점(W)·기회(O)·위협(T)을 도출하고 2×2 매트릭스로 시각화합니다.

5. **전략 보고서 자동 생성**
   분석 결과를 구조화된 마크다운/PDF 보고서로 자동 출력합니다. 시장 개황, 기업 포트폴리오, SWOT, 전략 종합, 투자·파트너십 의견을 포함합니다.

## 분석 대상

| 기업 | 설명 |
|------|------|
| LG에너지솔루션 | 한국 대표 배터리 제조사, NCM·파우치형 강점 |
| CATL | 글로벌 1위 배터리 제조사, LFP·각형 배터리 주도 |

## 최종 산출물

- `report_YYYYMMDD_HHMMSS.md` — 마크다운 전략 보고서
- `report_YYYYMMDD_HHMMSS.pdf` — PDF 전략 보고서

보고서 구성:
1. SUMMARY (핵심 인사이트 3~5줄)
2. 시장 현황 (글로벌 시장 규모·트렌드·경쟁 구도)
3. 기업 포트폴리오 & 기술 역량
4. 전략 비교 & SWOT
5. 종합 분석 (포지셔닝 차이·시장 전망·투자 의견)

## 기술 스택

| 구성요소 | 기술 |
|---------|------|
| LLM | Claude Sonnet 4.6 (Anthropic) |
| 에이전트 오케스트레이션 | LangGraph, LangChain |
| 벡터 DB | Qdrant (자체 호스팅) |
| 임베딩 | BAAI/bge-m3 (1024차원) |
| 웹 검색 | Tavily API, GoogleNews |
| 문서 파싱 | PDFPlumber |
| 보고서 출력 | ReportLab |
| 설정 관리 | Pydantic Settings |
