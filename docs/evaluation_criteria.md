# 평가 기준 (Evaluation Criteria)

에이전트가 분석 시 사용하는 판단 기준 모음.
코드 상 위치: `battery_market_agent/agents/market_analysis_agent.py`, `swot_analysis_agent.py`, `tech_analysis_agent.py`, `company_comparison_agent.py`

---

## 1. SWOT 판단 기준

배터리 기업 SWOT 분석 시 각 항목을 어떤 기준으로 분류할지 정의한다.
코드 반영 위치: `swot_analysis_agent.py` — `STRENGTH/WEAKNESS/OPPORTUNITY/THREAT_CRITERIA`

### 1.1 Strengths (내부 강점)

경쟁사 대비 우위에 있거나, 기업이 실질적으로 잘 수행하고 있는 내부 역량.

| 분류 | 세부 평가 항목 |
|---|---|
| 기술 역량 | 배터리 셀 에너지 밀도·사이클 수명·안전성 지표 |
| 생산 역량 | 생산능력(GWh) 및 가동률 |
| 지식재산 | 핵심 특허 보유 수 및 기술 차별화 요소 |
| 고객 기반 | tier-1 OEM 고객사 확보 다양성 |
| 원가 경쟁력 | 셀 제조 원가($/kWh) 및 수율 |
| 공급망 | 수직계열화 수준 (원자재 조달 → 셀 → 팩) |
| R&D | R&D 투자 비율 및 차세대 기술(전고체 등) 개발 현황 |

### 1.2 Weaknesses (내부 약점)

내부적으로 부족하거나 개선이 필요한 영역. 경쟁사 대비 열위.

| 분류 | 세부 평가 항목 |
|---|---|
| 원자재 의존 | 리튬·코발트·니켈 조달의 특정 국가·공급사 집중도 |
| 수익성 | 영업이익률 및 수익성 압박 |
| 기술 편중 | 특정 배터리 화학(NCM vs LFP) 편중 리스크 |
| 지역 편중 | 단일 시장 매출 의존도 |
| 재무 | 부채비율 및 재무 건전성 |
| 고객 집중 | 단일 OEM 고객사 매출 집중도 |

### 1.3 Opportunities (외부 기회)

기업 외부 환경에서 성장에 유리하게 작용하는 요소.

| 분류 | 세부 평가 항목 |
|---|---|
| 정책·보조금 | 각국 전기차 보조금(IRA·CBAM 등) 수혜 가능성 |
| 신시장 | ESS(에너지저장장치) 시장 확대 |
| 기술 선점 | 차세대 배터리(전고체·LFP 고도화) 수요 선점 기회 |
| 지역 확장 | 신흥 시장(동남아·인도·중동) 진출 가능성 |
| 파트너십 | 전략적 파트너십·합작법인(JV) 기회 |

### 1.4 Threats (외부 위협)

기업 외부 환경에서 리스크 요인으로 작용하는 요소.

| 분류 | 세부 평가 항목 |
|---|---|
| 경쟁 | 중국 배터리 업체(BYD·CATL·CALB 등) 저가 공세 및 시장 잠식 |
| 원자재 | 리튬·코발트·니켈 가격 변동성 및 공급망 불안정 |
| 고객 내재화 | 주요 OEM 고객사 배터리 내재화(In-house) 전략 |
| 규제 | EU 배터리 규정·공급망 실사 의무화 등 규제 불확실성 |
| 기술 패러다임 | LFP 전환 가속에 따른 NCM 중심 전략 리스크 |
| 수요 | 경기침체·EV 수요 둔화에 따른 가동률 하락 |

## 2. 시장 분석 평가 기준

`market_analysis_agent`가 분석 결과를 작성할 때 각 항목을 어떤 기준으로 판단할지 정의한다.
코드 반영 위치: `market_analysis_agent.py` — `SYSTEM_PROMPT`

### 2.1 글로벌 배터리 시장 규모 및 성장률

| 평가 항목 | 긍정 신호 | 부정 신호 |
|---|---|---|
| 시장 규모 | 연간 시장 규모($GWh·$B) 명시, 전년 대비 성장률 제시 | 성장률 둔화, 수요 예측치 하향 조정 |
| 성장 전망 | IEA·BloombergNEF 등 공신력 있는 기관의 CAGR 인용 | 주요 기관 전망 엇갈림, 불확실성 증대 |
| 수요 드라이버 | EV 보급률·ESS 설치량 증가 근거 | EV 수요 둔화, 구매 보조금 축소 |

### 2.2 해당 기업의 시장 점유율 및 포지셔닝

| 평가 항목 | 기준 |
|---|---|
| 글로벌 점유율 | 해당 연도·분기 기준 점유율(%) 및 전년 대비 증감 명시 |
| 순위 변동 | 상위 5개사 내 순위 추이 (상승·유지·하락) |
| 고객 포트폴리오 | tier-1 OEM 고객사 수, 신규 수주 공시 |
| 경쟁 압박 지표 | 중국 업체 점유율 확대에 따른 압박 수준 |

### 2.3 주요 지역별 수요 동향

| 지역 | 핵심 평가 항목 |
|---|---|
| 중국 | NEV 판매량, CATL·BYD 내수 점유율, 보조금 정책 변화 |
| 유럽 | EV 판매량, CBAM·배터리법 규제 영향, 현지 기가팩토리 투자 현황 |
| 북미 | IRA 세액공제(AMPC) 수혜액, 완성차 업체 EV 전략 변화 |
| 공통 | 지역별 수요 성장률 격차, 지역 편중 리스크 |

### 2.4 원자재 가격 동향

| 원자재 | 평가 기준 |
|---|---|
| 리튬 (탄산리튬) | 현재 가격($/톤), 전분기 대비 변동률, 가격 하락이 원가에 미치는 영향 |
| 코발트 | 공급 집중도(DRC 의존도), 가격 변동성 수준 |
| 니켈 | LME 가격 추이, 고니켈 양극재 원가 영향 |
| 종합 판단 | 원자재 가격 하락 → 원가 개선 기회 / 상승 → 수익성 압박 여부 |

### 2.5 최신 뉴스 및 산업 이슈

에이전트는 다음 유형의 이슈를 반드시 포함해야 한다.

| 이슈 유형 | 포함 여부 기준 |
|---|---|
| 긍정 이슈 | 대형 수주·파트너십·신제품 출시·신공장 착공 |
| 부정 이슈 | 실적 쇼크·적자 전환·고객 이탈·리콜·소송·신용등급 하향 |
| 정책 이슈 | 보조금 변화·무역 장벽·공급망 실사 의무화 |
| 기술 패러다임 | LFP 전환 가속, 전고체 상용화 일정 변화 |
| 애널리스트 시각 | 목표주가 변경, 투자의견 상향·하향 근거 |

## 3. 기술 분석 평가 기준

`tech_analysis_agent`가 RAG 문서를 기반으로 기술 역량을 평가할 때 사용하는 항목별 기준.
코드 반영 위치: `tech_analysis_agent.py` — `TECH_CRITERIA`, `SYSTEM_PROMPT`

### 3.1 배터리 셀 기술 및 에너지 밀도

| 평가 항목 | 기준 | 부정 신호 |
|---|---|---|
| 에너지 밀도 | 셀 Wh/kg·Wh/L 수치 및 경쟁사 대비 수준 | 경쟁사 대비 낮거나 개선 정체 |
| 사이클 수명 | 80% 용량 유지 기준 충방전 횟수 | 조기 열화, 보증 클레임 증가 |
| 급속 충전 | C-rate 및 충전 시간(10~80%) | 고속 충전 시 열화 문제 |
| 안전성 | 열폭주 방지 기술 보유 및 인증 현황 | 리콜·화재 사고 이력 |
| 화학 포트폴리오 | NCM·LFP·NCMA·나트륨이온 다양성 | 단일 화학 계열 의존 |

### 3.2 양극재·음극재 기술 수준

| 평가 항목 | 기준 | 부정 신호 |
|---|---|---|
| 양극재 니켈 함량 | 고니켈 NCM 9xx·NCMA 적용 여부 | 저니켈 계열에 머무는 경우 |
| 양극재 내재화 | 자체 개발·생산 비율(%) | 외부 조달 100% 의존 |
| 실리콘 음극재 | Si 함량(%) 및 팽창 제어 기술 | Si 도입 미흡, 팽창 문제 미해결 |
| 전해질 | 액체→반고체→고체 전환 개발 단계 | 기존 액체 전해질만 사용 |
| 소재 조달 리스크 | 공급망 다변화 수준 | 특정 국가(중국) 단일 의존 |

### 3.3 BMS(배터리 관리 시스템) 역량

| 평가 항목 | 기준 | 부정 신호 |
|---|---|---|
| SOC·SOH 정확도 | 추정 오차율(%) | 오차 큰 경우 안전·보증 리스크 |
| AI 기반 BMS | 머신러닝 적용 여부 및 기능 | 전통적 알고리즘만 사용 |
| OTA 지원 | 무선 소프트웨어 업데이트 가능 여부 | 미지원 시 서비스 경쟁력 열위 |
| 셀 밸런싱 | 능동·수동 밸런싱 기술 수준 | 수동 방식만 적용 |

### 3.4 생산 공정 및 원가 경쟁력

| 평가 항목 | 기준 | 부정 신호 |
|---|---|---|
| 셀 제조 원가 | $/kWh 수치 및 경쟁사 대비 수준 | CATL 대비 20% 이상 고원가 |
| 생산 수율 | %(불량 제외 정상 생산 비율) | 수율 저하·라인 중단 이슈 |
| 건식 전극 공정 | Dry Electrode 도입 여부 및 단계 | 습식 공정만 운영 |
| 스마트팩토리 | 자동화·AI 품질 관리 수준 | 수동 공정 비중 높음 |
| CAPEX 효율성 | GWh당 투자비 | 경쟁사 대비 과도한 투자비 |

### 3.5 차세대 기술 개발 현황

| 평가 항목 | 단계 구분 | 부정 신호 |
|---|---|---|
| 전고체 배터리 | 기초연구 → 시제품 → 파일럿 → 양산 로드맵 확정 | 로드맵 지연·미공개 |
| 4680 원통형 셀 | 개발·양산 준비 단계 | 경쟁사 대비 도입 지연 |
| 리튬메탈 음극 | 연구·개발 수준 | 상용화 일정 불명확 |
| 나트륨이온 | 개발 여부 | 미착수 시 신시장 대응 미흡 |
| 발표 vs 실제 | 공식 발표 일정과 실제 진행 간 괴리 여부 | 반복적 일정 지연 |

### 3.6 핵심 특허 및 기술 차별화

| 평가 항목 | 기준 | 부정 신호 |
|---|---|---|
| 특허 총수 | 보유 특허 수 및 분야별 분포 | 경쟁사 대비 현저히 적은 경우 |
| 핵심 분야 특허 | 소재·셀·BMS·공정 각 분야 핵심 특허 보유 여부 | 특정 분야 공백 |
| 라이선스 수지 | 라이선스 수입 > 지출이면 기술 우위 신호 | 수입 없이 지출만 발생 |
| 특허 분쟁 | 소송 이력 및 해결 여부 | 주요 특허 무효화·패소 이력 |

## 4. 기업 분석 Supervisor 평가 기준

`company_analysis_agent`(Supervisor)가 market·tech·swot 세 에이전트 결과를 종합할 때 적용하는 기준.
코드 반영 위치: `company_analysis_agent.py` — `_SUPERVISOR_PROMPT`

### 4.1 하위 에이전트 실행 완결성

| 검증 항목 | 완결 기준 |
|---|---|
| market_analysis_agent | 시장 규모·성장률·지역별 동향·원자재 가격·최신 이슈 5개 항목 모두 존재 |
| tech_analysis_agent | 셀 기술·양극재·BMS·생산 공정·차세대 기술·특허 6개 영역 모두 존재 |
| swot_analysis_agent | 2×2 SWOT 행렬(S·W·O·T 각 최소 3항목) 반환 |
| 실행 순서 | market → tech → swot 순서 준수 |

### 4.2 종합 보고서 품질 기준

Supervisor가 세 하위 결과를 통합한 최종 `company_report`에 적용한다.

| 평가 항목 | 기준 | 부정 신호 |
|---|---|---|
| 통합성 | 시장·기술·SWOT 결과가 서로 참조되어 연결 서술 | 세 분석이 단순 나열, 상호 참조 없음 |
| 수치 인용 | 핵심 지표(점유율·원가·에너지 밀도)에 구체적 수치 포함 | 수치 없이 정성적 서술만 |
| 균형성 | 강점·약점·기회·위협이 균형 있게 기술 | 긍정 내용 일색, 리스크 누락 |
| 전략적 시사점 | 투자·파트너십·사업 관점의 시사점 포함 | 현황 나열에 그치고 시사점 부재 |
| 구조화 | 섹션 구분, 표·소제목 활용 | 비구조적 자유 서술 |

### 4.3 에이전트 간 일관성 검증

| 검증 항목 | 기준 |
|---|---|
| 시장–SWOT 일관성 | 시장 분석에서 도출된 위협 요인이 SWOT T 항목에 반영 |
| 기술–SWOT 일관성 | 기술 강점·한계가 SWOT S·W 항목과 일치 |
| 출처 정합성 | market_sources에 기록된 출처가 보고서 내 주장과 연결 |

## 5. 기업 비교 평가 기준

`company_comparison_agent`가 LG에너지솔루션 vs CATL을 5개 차원으로 비교할 때 사용하는 기준.
코드 반영 위치: `company_comparison_agent.py` — `COMPARISON_CRITERIA`, `_COMPARISON_PROMPT`

### 5.1 기술 전략

| 평가 항목 | 비교 포인트 |
|---|---|
| 배터리 화학 포트폴리오 | NCM 고니켈·NCMA·LFP 비중 및 화학 전환 대응 속도 |
| 폼팩터 전략 | 파우치·각형·원통형(4680) 채택 비율 및 방향성 |
| 에너지 밀도 | 셀 Wh/kg 수치 직접 비교 |
| 차세대 기술 준비도 | 전고체 개발 단계(연구→시제품→파일럿→양산 로드맵) 비교 |
| 특허 역량 | 보유 특허 수 및 핵심 기술 영역 커버리지 비교 |

### 5.2 지역 전략

| 평가 항목 | 비교 포인트 |
|---|---|
| 중국 | 현지 생산능력(GWh), 중국 OEM 고객 비중 |
| 유럽 | 기가팩토리 투자 규모·일정·현지 고객 확보 현황 |
| 북미 | IRA 세액공제(AMPC) 수혜 규모, 현지 JV 현황 |
| 신흥 시장 | 동남아·인도·중동 진출 계획 구체성 |
| 지역 편중 리스크 | 단일 지역 매출 의존도 비교 |

### 5.3 고객 전략

| 평가 항목 | 비교 포인트 |
